## Student Performance Predictor

# Problem Statement:
Teachers and students cannot easily identify who may perform poorly in exams. Academic performance depends on factors like attendance, study hours, previous marks, and assignments. Build an AI system that predicts student performance using these factors.

# Goal:
Predict student exam performance using machine learning.

# Input:
 
 Study hours
 Attendance percentage
 Previous marks
 Assignment completion (%)
 Internal test scores
 
# Output:
 
 Predicted final marks or
 Performance category (High / Medium / Low)
